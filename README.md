# SOLAR-TRAKER
Este proyecto es un seguidor solar de doble eje desarrollado con ESP32 y MicroPython. Procesa cuatro sensores LDR para orientar un panel hacia la luz mediante señales PWM: controla un servo de 180° en el eje vertical por ángulos y uno de 360° en el horizontal por tiempo. Usa SoftI2C para mostrar datos en una LCD en tiempo real.
